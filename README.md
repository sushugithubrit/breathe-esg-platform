# Breathe ESG Data Ingestion & Analytics Platform

## Overview

This project is a full-stack ESG data ingestion and analytics platform built using Django REST Framework and React.

The platform enables analysts to upload enterprise sustainability datasets, automatically process records, generate analytics, identify suspicious entries, and calculate ESG compliance metrics.

## Features

- Dynamic CSV ingestion
- Automatic schema detection
- Analytics dashboard
- Suspicious record detection
- ESG compliance scoring
- Interactive charts and visualizations
- REST API integration
- Support for multiple dataset structures
